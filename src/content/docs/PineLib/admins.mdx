Welcome to the PineLib documentation for server administrators! This guide is fairly short as PineLib is designed to blend into the background with zero configuration.

## Installation
To install PineLib on your server, simply download the latest version of the plugin from [Modrinth](https://modrinth.com/plugin/pinelib) and place the JAR file into your server's `plugins` directory. PineLib is compatible with Paper and its forks (since v1.3.0 spigot support was dropped), so make sure your server is running a compatible version of Minecraft.

## Dependencies
There are no required dependencies for PineLib, but some features may require additional libraries. For example, if you want to use anvil menus, you may need to install ProtocolLib as well. However, most features of PineLib are self-contained and do not require any additional plugins.
